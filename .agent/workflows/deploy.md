---
description: Deploy changes to Vercel by pushing to Github
---

1. Check git status to see pending changes
   `git status`

2. Stage all changes (or specific files if preferred)
   `git add .`

3. Commit changes with a descriptive message
   `git commit -m "Update site content and code"`

// turbo
4. Push to main branch to trigger Vercel deployment
   `git push origin main`
